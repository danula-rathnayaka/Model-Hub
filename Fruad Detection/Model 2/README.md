# Fraud Detection Model 1

**Data Source = https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud**

## PreProcessing

Apply SMOTE to treat class imbalance.

Use standard scaler in preprocessing to scale the X smote

## Model Training 

Developed using XG boost as the algorithm

### About XG boost :

XGBoost (eXtreme Gradient Boosting) is an optimized and scalable open-source machine learning library that implements gradient boosted decision trees
